Chetan Sharma: Quantitative Finance Portfolio

This site presents a collection of advanced quantitative finance projects built to explore market microstructure, portfolio optimisation, derivatives pricing, risk management, and machine learning. Each project is fully documented with code, results and visualisations.

Interactive Dashboard

Explore aggregated results and metrics in the interactive dashboard, including cumulative returns, efficient frontiers, a correlation heatmap and a risk metrics table: Open the Dashboard.

Project Highlights

  • Market microstructure & execution: Limit order book simulation, high-frequency C++ engine, Avellaneda–Stoikov quoting, Almgren–Chriss optimal execution, Kalman filter pairs trading.
  • Portfolio construction & risk: Mean–variance and risk parity optimisation, factor model regression, GARCH volatility forecasting, VaR/CVaR simulations.
  • Derivatives & stochastic volatility: Black–Scholes and Greeks calculators, Heston and rough Bergomi models, Monte Carlo option pricing, implied volatility surface.
  • Machine learning & RL: Sentiment analysis of financial news, reinforcement learning trading agent, backtesting engine.

For full descriptions and instructions, see the project portfolio README in this repository.
